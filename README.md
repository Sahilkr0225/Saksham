# 🏫 Saksham — School Management Platform

> A production-grade, modular School Management Platform for medium-sized schools (500–2000 students).

---

## 📖 About
Saksham eliminates fragmented school administration by providing a centralized, role-aware platform for Principals, Teachers, Parents, Students, and Office Staff.

## 🗺️ Project Status
See [docs/PROJECT_STATUS.md](docs/PROJECT_STATUS.md) for the full module-by-module progress tracker.

## 📐 Architecture
- **Backend:** Node.js + Express.js (REST API)
- **Frontend:** Next.js 14 (App Router)
- **Database:** PostgreSQL + Redis
- **Auth:** JWT + Refresh Token rotation
- **Deployment:** Docker + Docker Compose

## 📦 Modules (20 Core)
Authentication · RBAC · Admissions · Students · Parents · Teachers · Academic Structure · Attendance · Homework · Exams · Fees · Notices · Events · Notifications · Documents · Reports · Administration · Audit Logs · Gallery · Public Website
